Customer Segmentation & Churn Prediction

Segmented 9,164 e-commerce customers and predicted 30-day purchases with 92% accuracy to drive targeted marketing.

← Back to Portfolio

🔍 Project Overview

Analyzed purchase histories (2014–2021) to:

  • Cluster customers via K-means on RFM + PCA features.
  • Build XGBoost (plus baseline models) to predict next-30-day purchases.

🌟 Key Findings

  • Identified five distinct segments (high-value vs. at-risk groups).
  • XGBoost achieved ROC AUC 0.98 & 92% accuracy.
  • Recommended targeted retention offers and upsell campaigns.

🛠 Methods & Tools

  • Clustering: K-means with RFM + PCA inputs
  • Modeling: Naive Bayes, Logistic Reg., Decision Tree, Random Forest, XGBoost (10-fold CV)
  • Stack: Python, pandas, scikit-learn, matplotlib

🚀 Recommendations

  1. Create custom offers for at-risk clusters.
  2. Run upsell campaigns to high-value segments.
  3. Build an RFM dashboard for ongoing monitoring.
